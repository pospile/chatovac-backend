Advance security package

Okamžitě po připojení na socket server se spustí advance security package. To znamená, že se ihned po spuštění socketů musí uživatel dotázat na socket login. (Tento login je jiný, než login přes REST API - viz níže)

login

  • name (jméno uživatele)
  • token (token získaný při REST loginu)
  • device_id (device_id které odpovídá tokenu - získáno při REST loginu)

login_resp: odpověď: jako odpoveď dorazí json s errorem: true/false a dobou do vypršení tokenu

Odeslání požadavku na login musí proběhnout do dvou vteřin od připojení na server. Je třeba naslouchat na odpověď login_resp, kam přijde odpověď o případném úspěchu či neúspěchu.
